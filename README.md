# 🍕 Pizza Paradise - Online Pizza Website

A modern and responsive pizza website designed to showcase delicious pizzas, burgers, breakfast items, and beverages with an attractive user-friendly interface.

This project was created using **HTML and CSS** with a focus on clean design, responsive layouts, and an enjoyable food browsing experience.

---

## 📌 Features

✨ **Beautiful Landing Page**
- Attractive hero section with food visuals
- Catchy headings and call-to-action buttons

🍕 **Pizza Menu**
- Multiple pizza categories
- Images with descriptions and pricing
- Clean card-based layout

🍔 **Food Categories**
- Pizzas
- Burgers
- Breakfast items
- Drinks

📱 **Responsive Design**
- Optimized layout for desktop, tablet, and mobile screens

🎨 **Modern UI Design**
- Attractive color scheme
- Hover effects
- Organized sections for better user experience

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome Icons
- Responsive Web Design
